props:
  - name: variant
    type: '"neutral" | "danger" | "warning" | "success" | "merged"'
    default: '"neutral"'
    description: Semantic color variant.
  - name: color
    type: string
    description: Hex color (e.g. "#d73a4a"). Overrides variant. Background and foreground are computed automatically for readability.

## Custom Color

Pass a `color` prop with any hex value. The badge computes a tinted background and adjusts the foreground color for readability. Used for dynamic data like GitHub issue labels.
