Clean, minimal, rounded. Monochrome with intent. 

## Getting Started

This is the component library for twitbruv. Components are built with React 19, Base UI primitives, and Tailwind CSS v4.

## Design Principles

- **Minimal** — every element earns its place
- **Rounded** — soft corners, friendly surfaces
- **Monochrome** — color is used with purpose, not decoration
